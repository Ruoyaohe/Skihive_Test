## 🎯 动画技术细节

### 使用的动画库
- **Framer Motion**: 所有动画效果
- **Spring 动画**: 自然弹性效果
- **Ease 缓动**: 平滑过渡

### 关键动画参数

1. **Spring 动画**
   ```typescript
   {
     type: 'spring',
     stiffness: 200-400,  // 弹性强度
     damping: 15          // 阻尼
   }
   ```

2. **过渡时间**
   - 快速交互: 200ms
   - 标准动画: 300-500ms
   - 复杂动画: 800-1500ms

3. **缩放效果**
   - 悬停: 1.02 - 1.1
   - 点击: 0.95 - 0.98

## 🔄 状态管理

### 登录表单
- `email`: 邮箱输入
- `password`: 密码输入
- `isLoading`: 加载状态
- `agreed`: 协议同意状态

### 开始飞行
- `isDragging`: 拖动状态
- `hasStarted`: 完成状态
- `x`: 拖动位置 (MotionValue)
- `progress`: 进度百分比 (Transform)

### 装配页面
- `selectedComponents`: 已选组件数组
- `isAnalyzing`: AI分析状态
- `currentStep`: 当前装配步骤
- `isAssembling`: 装配进行中

### 环境选择
- `selectedEnv`: 选中的环境ID
