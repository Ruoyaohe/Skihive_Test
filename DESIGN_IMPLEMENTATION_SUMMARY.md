## 🎯 动画技术

### 使用的库
- **Framer Motion**: 所有动画效果
- **Spring 动画**: 自然弹性效果
- **Transform**: 位置和尺寸变换

### 关键动画类型
1. **淡入淡出**: `opacity: 0 → 1`
2. **缩放**: `scale: 0 → 1` 或 `scale: 1 → 1.1`
3. **位移**: `y: -20 → 0` 或 `x: 0 → maxDrag`
4. **旋转**: `rotate: 0 → 360` (无限循环)
5. **颜色过渡**: 边框和背景色变化
